Gerenciador de Configurações (config)

Manager

Módulo responsável por gerenciar a leitura dos arquivos YAML de configuração.

Ele cria os diretórios necessários e fornece funções para que as classes de configuração sejam utilizadas pelos respectivos componentes.

class fraud_detection.config.manager.ConfigurationManager(config_filepath=PosixPath('config/config.yaml'), params_filepath=PosixPath('config/params.yaml'), schema_filepath=PosixPath('config/schema.yaml'))[código-fonte]

Base: object

Classe responsável por carregar e gerenciar configurações a partir de arquivos YAML.

Lê os arquivos de configuração, cria diretórios e fornece métodos para que os componentes do pipeline acessem suas configurações.
