Pipelines (pipeline)

Pipeline de predição (prediction)

Módulo do pipeline utilizado para predição.

Pensado inicialmente para processamento de apenas um dados recebido.

A partir do dado recebido carrega o modelo, aplica o pré-processamento adequado aos dados de entrada e devolve a probabilidade da classe.

class fraud_detection.pipeline.prediction.PredictionPipeline[código-fonte]

Base: object

Class contendo pipeline de predição, submete os dados de entrada ao mesmo pipeline de transformação ajustado na etapa de pré-processamento dos dados de treino.

predict(data)[código-fonte]

Função que realiza a predição dos dados, também invoca o método para pré-processar os dados inseridos.

Parâmetros:

data (pd.DataFrame) – Dados de entrada para predição.

Retorna:

  • prediction int: Classe a qual os dados foram classificados

  • prediction_proba list: Lista de probabilidades das classes

Tipo de retorno:

tuple

transform_input_data(data)[código-fonte]

Aplica o pipeline de pré-processamento nos dados a serem usados para predição. O pipeline possui informações de Imputers e Encodings salvos a partir apenas dos dados de treino.

Parâmetros:

data (pd.DataFrame) – DataFrame de linha única contendo dados a serem submetidos ao modelo.

Retorna:

Dados transformados para aplicação ao modelo treinado.

Tipo de retorno:

pd.DataFrame
